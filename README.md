# Emeka Logistics — Website

A professional freight and shipping logistics website built with pure HTML, CSS, and JavaScript. Navy & Gold theme.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage — hero, services preview, stats, testimonials |
| `services.html` | Full services list |
| `tracking.html` | Shipment tracking tool |
| `about.html` | Company info & values |
| `contact.html` | Quote request form |

## Project Structure

## How to Run Locally

1. Open the folder in VS Code
2. Install the **Live Server** extension
3. Right-click `index.html` → **Open with Live Server**

## Deploy to GitHub

```bash
git init
git add .
git commit -m "Initial commit — Emeka Logistics website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/emeka-logistics.git
git push -u origin main
```

## Customization

- Update contact details in each page's footer and `contact.html`
- Replace placeholder phone/email with real info
- Add your own logo image in the navbar if desired
- Update color variables in `css/style.css` under `:root`

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Playfair Display + Barlow)
- No frameworks or build tools needed
